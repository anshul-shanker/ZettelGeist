# ZettelGeist

A handy web-application for creating and maintaining notes

## Configurations

  - Ruby -v `3.0.0p0`
  - Rails -v `6.1.1`
  - PostgreSQL -V `13.1`

## Steps to run the Application (localy):

  - `bundle install`
  - `rails db:migrate`
  - `rails server`
  - `localhost:3000`
